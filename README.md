# Pokegallery 

Simple App to allow user to search any Pokémon with details on their type, Pokédex number, as well as their sprite. *Going to continue to add content over time.

# ⚙️ Technologies Used

-HTML 
-CSS 
-JS
-jQuery
-[Pokemon API](https://pokeapi.co/)
